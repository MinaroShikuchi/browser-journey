# Browser Journey - Privacy Policy

**Last Updated: November 20, 2024**

## Introduction

Browser Journey ("we", "our", or "the extension") is committed to protecting your privacy. This Privacy Policy explains how Browser Journey collects, uses, and safeguards your data.

## Data Collection

### What Data We Collect

Browser Journey collects the following data **locally on your device**:

1. **Browsing History Data**
   - URLs of websites you visit
   - Page titles
   - Domain names
   - Visit timestamps
   - Navigation transitions between pages

2. **Tab Information**
   - Currently open tabs
   - Tab creation and closure events
   - Tab relationships (which tab opened which)

3. **User Preferences**
   - Filter settings (date ranges, search terms)
   - Pinned journeys
   - UI preferences (panel sizes, collapsed menus)

### How We Collect Data

Data is collected automatically as you browse the web through:
- Chrome's `tabs` API for tab monitoring
- Chrome's `webNavigation` API for navigation tracking
- Chrome's `history` API for historical data access

## Data Storage

### Local Storage Only

**All data is stored exclusively on your local device** using Chrome's local storage API. We do NOT:
- Transmit data to external servers
- Upload data to the cloud
- Send data to third parties
- Use remote databases

### Data Retention

- Data is retained indefinitely on your device until you manually delete it
- You can clear all data at any time using the "Clear History" button
- Uninstalling the extension removes all stored data

## Data Usage

### How We Use Your Data

Your browsing data is used solely for:
1. Creating interactive visualizations of your browsing journeys
2. Generating statistics about your browsing patterns
3. Enabling journey management features (export, import, delete)
4. Highlighting currently active tabs
5. Providing AI-generated journey titles (processed locally using Chrome's Built-in AI)

### What We DON'T Do

We do NOT:
- Sell your data
- Share your data with third parties
- Use your data for advertising
- Track your activity for analytics
- Transmit your data off your device
- Use your data for any purpose other than local visualization

## Data Sharing

**We do not share your data with anyone.** All data remains on your local device.

### User-Initiated Sharing

You may choose to:
- Export journey data as JSON files to share with others
- Import journey data from JSON files shared by others

These actions are entirely voluntary and under your control.

## Data Security

### Security Measures

- All data is stored in Chrome's secure local storage
- No network transmission of browsing data
- No external API calls with user data
- Extension code is open source and auditable

### Your Control

You have complete control over your data:
- **View**: See all collected data in the visualization
- **Export**: Download your data as JSON files
- **Delete**: Clear all history with one click
- **Manage**: Delete individual journeys

## Third-Party Services

### Chrome Built-in AI

Browser Journey may use Chrome's Built-in AI API to generate journey titles. This processing:
- Happens locally on your device
- Does not send data to external servers
- Is optional (falls back to domain names if unavailable)
- Follows Chrome's AI privacy policies

### No Other Third Parties

We do not use:
- Analytics services
- Advertising networks
- External APIs
- Cloud storage services
- Third-party tracking

## Children's Privacy

Browser Journey does not knowingly collect data from children under 13. The extension is designed for general browsing visualization and is not targeted at children.

## Your Rights

You have the right to:
- **Access** your data through the extension interface
- **Export** your data as JSON files
- **Delete** your data at any time
- **Control** what data is collected through Chrome's extension permissions

## Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected in the "Last Updated" date. Continued use of the extension after changes constitutes acceptance of the updated policy.

## Permissions Explained

### Why We Need Permissions

- **tabs**: Monitor browsing activity and detect open tabs
- **storage**: Save data locally on your device
- **history**: Access browsing history for visualization
- **webNavigation**: Track cross-tab navigation
- **alarms**: Schedule periodic cleanup tasks
- **host_permissions (<all_urls>)**: Track visits across all websites

All permissions are used exclusively for local data processing.

## Data Portability

You can export all your data at any time:
1. Click the export button in the extension
2. Receive a JSON file with all your browsing journey data
3. Import this data into another installation or share with others
